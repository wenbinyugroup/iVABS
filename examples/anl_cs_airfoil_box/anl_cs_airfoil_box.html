Analysis of a single cross-section#

Suppose that we want to carry out a single cross-sectional analysis. The task is to carry out a quick cross-sectional analysis of a specific cross-sectional design. Specifically, in this example, we want to set values to the locations of spar webs and get the torsional and bending stiffness (\(GJ\), \(EI_{yy}\), \(EI_{zz}\)).

Running#

ivabs main.yml --mode 1

Results#

Analysis results can be found in the file main.out.
