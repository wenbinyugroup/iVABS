iVABS IntroductionÂ¶

iVABS (integrated VABS) package bundles PreVABS + VABS + gmsh + msgpi + GEBT + Dakota(optional) . PreVABS is a parametrized composite design tool. VABS is a commercial code for cross-sectional property analysis. msgpi is a Python interface for VABS. GEBT is a beam structural analysis tool. PreVABS, VABS, msgpi, and GEBT are developed in Prof. Yuâ€™s group. Gmsh is an open source CAD software. Dakota is an open source tool for optimization developed by Sandia national lab.
