Problem descriptionÂ¶

This example has a cross section of a simple circular shape with radius \(r=10\) m. This cross section geometry can be defined easily by a center and a radius. Material properties are given in Table 16. The layup is defined using the stacking sequence code \([\pm 45_2/0_2/90]_{2s}\). The result is given in Table 18. Complete input files can be found in examples\ex_tube\, including tube.xml and materials.xml.
