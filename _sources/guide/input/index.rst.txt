Main Input
========================



The main input file uses the format `YAML <https://yaml.org/>`_.

Overall layout of the main input file:

..  code-block:: yaml

    version: "0.9"
    structure:
        # Global structure definition block
    sg:
        # SG block
    function:
        # Function block
    analysis:
        # Analysis steps block
    study:
        # Design study block
